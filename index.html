O que são Clients?

Clients são todos os tipos de aparelhos (Celular, relogio, geladeira ...) que solicitam informações para algum Servers, por exemplo, abrir um site no navegador, fazer download de uma musica ou dar play em um video, essas solicitações são enviadas para o servidor que devolve a informação solicitada.

O que são Servers?

Já os Servers são computadores usados para armazenamento de dados e existem tanto servidores remotos quanto locais, geralmente os servidores são computadores mais robustos ounão por terem que se conectar com vários Clients fazendo com que tenham que processar muita informação simultaneamente.

Como é uma estrutura básica de um HTML?

A estrutura básica do HTML basicamente não muda, geralmente editores de código tem atalhos para criar essa estrutura. Sendo ela as tags:
<html></html> - Todo documento HTML deve iniciar e finalizar com essa tag.
<head></head> - Tag que determina o cabeçalho do documento, as informações aqui não são visíveis para o usuário do documento, mas isso não significa que pode colocar informações sigilosas as informações colocadas no HEAD podem ser vistas por outros meios.
<title></title> - Essa tag diz qual é o título da página, é o nome que aparece na aba/janela/guia do seu navegador.
<body></body> - Essa tag é o documento em si as informações legiveis*ao usuário.

*Não só texto como imagens, musicas, qualquer conteúdo multimídia e etc.
DOCTYPE apesar de não ser uma tag HTML ele sempre deve estar presente! Ele é uma instrução para os programas que podem ler seu site.

Atríbutos das Tags

Existe uma infinidade de atributos e eles são divididos em dois tipos, os globais aceitos por todas as tags como, por exemplo: id, class e style e também existem tags com propriedades específicas como type ou src.

Voltar

Tá, mas para que serve?

Os atributos mudam o modo como o navegador vai apresentar uma tag para o usuário.

Ex:
A tag input sendo usada com atributo color <input type="color"> fica assim: onde o usuário pode escolher a cor que quiser, mas também pode ser usada, por exemplo, com atributo text <input type="text"> que é mostrado uma caixa onde usuário pode escrever o que quiser. Às duas são tags inputs, mas estão sendo usadas de formas completamente diferentes.
